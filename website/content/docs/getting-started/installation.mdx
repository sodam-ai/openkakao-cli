# Installation

## Homebrew

```bash
brew tap JungHoonGhae/openkakao
brew install openkakao-cli
```

Upgrade later with:

```bash
brew update && brew upgrade openkakao-cli
```

## Build from source

Requires Rust `>= 1.75`.

```bash
git clone https://github.com/JungHoonGhae/openkakao-cli.git
cd openkakao/openkakao-cli
cargo install --path .
```

## Verify

```bash
openkakao-cli --version
```

## Shell completions

```bash
# zsh
openkakao-cli completions zsh >> ~/.zfunc/_openkakao-cli

# fish
openkakao-cli completions fish > ~/.config/fish/completions/openkakao-cli.fish

# bash
openkakao-cli completions bash >> ~/.bash_completion
```

## Before you continue

Make sure the KakaoTalk macOS app is installed and already logged in. OpenKakao depends on local app state and will not bootstrap a fresh Kakao account session by itself.
