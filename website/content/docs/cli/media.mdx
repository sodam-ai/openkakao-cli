---
title: download
description: Download media attachments from messages.
---

## Usage

```bash
openkakao-cli download <chat_id> <log_id> [OPTIONS]
```

## Options

| Flag | Description | Default |
|------|-------------|---------|
| `-o, --output-dir <DIR>` | Output directory | `downloads` |

## Example

```bash
# Download media from a specific message
openkakao-cli download 900000000000001 9000000000000000001

# Custom directory
openkakao-cli download 900000000000001 9000000000000000001 -o ./media
```

Files are saved as `{output_dir}/{chat_id}/{log_id}_{filename}`.

## Finding the log_id

The `log_id` is shown in JSON output:

```bash
openkakao-cli read <chat_id> --json | jq '.[] | select(.type != 1) | {logId, type, message}'
```

## Auto-Download

For automatic media download, use the `watch` command:

```bash
openkakao-cli watch --download-media --download-dir ./media
```

See [watch](/docs/cli/watch) for details.
